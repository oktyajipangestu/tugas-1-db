1. Apa yang dimaksud dengan database?
database adalah kumpulan data yang terorganisasi

2. Apa singkatan dan pengertian dari RDBMS?
RDBMS atau Relational Database Management System merupakan aplikasi database yang menggunakan prinsip relasional database model, 

3. Sebutkan 5 cara apa saja yang dipakai dalam penyimpanan data pada database model!
- Flat Model : Menyimpan data tanpa aturan dan cara penulisan tertentu.  
- Hierarchical Model : Data terorganisasi dari atas ke bawah.  
- Network Model : Pengembangan dari hierarchical model.  
- Relational Model  : Menyusun data ke dalam bentuk tabel yang saling terhubung.
- Object Oriented Model : Menggunakan pendekatan object dalam menyimpan data.    

4. Sebutkan 6 aplikasi RDBMS selain MySQL!
Oracle, Sybase, Microsoft Access, Microsoft SQL Server, PostgreSQL dan juga MariaDB

5. Sebutkan 5 keunggulan MySQL sebagai RDBMS!
- Speed : Memiliki kecepatan yang lebih
- Reliability
- Scalability : MySql mampu mengelola data yang besar dan kompleks
- User Friendly : kemudahan instalasi dan mudah dipelajari
- Portability and Standard Compliance : dapat berpindah dari suatusu sitem ke sistem lainnya misal, window ke mac
- Multiuser Support : dapat diakses bersamaan
- Internationalization :mendukung berbagai macam bahasa
- Wide Application Support : berbagai bahasa pemrograman yang mendukung
- Open source code : bebas digunakan
